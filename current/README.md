# current/ — live class materials

One folder per class. Each holds that class’s **syllabus, rubrics, assignment prompts, and readings**.

The seven required OLL core courses are pre-created. Add elective folders as you enroll (see `_electives/`). When a new class starts, drop its syllabus and prompts into its folder and tell Claude — it reads from here to know what’s assigned and what’s due.

**Required core (21 cr):**
| Folder | Course |
|---|---|
| HOL-6700-... | Human Behavior and Learning in Organizations |
| HOL-6701-... | Adult Learning |
| HOL-6702-... | Organizational Change |
| HOL-6704-... | Leadership in Organizations |
| HOL-6709-... | Leadership Development |
| HOL-6721-... | Assessing the Impact of Organizational Change (research methods) |
| HOL-6746-... | Work Groups and Teams in Organizations |

Plus **9 elective credits** — see `_electives/README.md`.
